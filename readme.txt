njTabs - это jQuery/j library плагин, реализующий функционал табов на сайте.

Демо - http://nejik.github.io/njTabs/

Из особенностей можно упомянуть:

MIT
ie7+ (необходим jQuery)
куча настроек
управление с помощью data аттрибутов
автоматическую инициализацию
гибкую систему анимаций (использующую css3)
поддержка ивентов
используемые jQuery методы
независимость от стилей (вы стилизуете все как вам нужно, затем навешиваете плагин и он делает всю логику)
